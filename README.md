# User Retention Prediction

## 📊 Описание проекта
Прогнозирование возврата пользователей в мобильном приложении на основе поведенческих признаков.

**Задача:** Бинарная классификация
- `1` — пользователь вернется в приложение
- `0` — пользователь не вернется

**Метрика:** ROC-AUC

## 🎯 Цель
Предсказать, вернется ли пользователь в приложение в ближайший период, чтобы вовремя запускать механики удержания.
